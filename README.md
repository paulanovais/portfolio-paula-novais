# Portfólio Online — Paula Aparecida Novais Silva

Projeto criado para a atividade prática de Fundamentos da Programação Web.

## Estrutura
- `index.html`: página única com navegação por âncoras.
- `style.css`: estilos em CSS puro, sem frameworks.
- `script.js`: menu responsivo, tema claro/escuro e validação do formulário.
- `assets/`: imagens, logos e vídeos do portfólio.

## Como publicar no GitHub Pages
1. Crie um repositório público no GitHub.
2. Envie todos os arquivos deste projeto para o repositório.
3. Vá em Settings > Pages.
4. Em Source, selecione a branch `main` e a pasta `/root`.
5. Salve e aguarde o GitHub gerar o link publicado.

## Atenção
Depois de publicar, coloque no PDF de entrega:
- Link do site publicado.
- Link do repositório público.
- Prints do site aberto no navegador, com o endereço do GitHub Pages aparecendo na barra.
